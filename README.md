# 📊 Smart Student Performance System

## 📌 Description
This project is a Python-based system designed to analyze student scores.

The system can:
- Accept student scores from user input
- Calculate the average score
- Find the minimum and maximum scores
- Analyze pass/fail status of each student

---

## 👥 Team Members & Contributions

This project was developed as a group task. Each student implemented one function:

- 👤 Student 1 → `input_scores()`
- 👤 Student 2 → `calculate_average(scores)`
- 👤 Student 3 → `find_min_max(scores)`
- 👤 Student 4 → `analyze_pass_fail(scores)`

---

## 🧠 Features

- ✅ Input validation (only numbers allowed)
- ✅ Handles empty input (edge case)
- ✅ Pass/Fail classification
- ✅ Clean modular structure
- ✅ Uses core programming concepts:
  - Variables
